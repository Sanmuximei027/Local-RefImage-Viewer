# Local-RefImage-Viewer (本地参考图查看器)

> 一个轻量、极速、专为创作者与画师打造的本地图片参考管理工具。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node](https://img.shields.io/badge/node-%3E%3D14.0.0-brightgreen.svg)

**Local-RefImage-Viewer** 是一个基于 Node.js 构建的本地图库可视化工具。它不会将你的任何文件上传到云端，所有操作均在本地极速完成。它支持强大的 **Windows 右键菜单集成**，并且针对画师看图做出了众多交互优化（如极速缩略图、沉浸式拖拽查看等）。

## ✨ 核心特性

- 🔒 **绝对隐私**：纯本地运行，不联网，不上传，不破坏原文件夹结构。
- ⚡ **极速性能**：引入企业级 `sharp` 图像处理库，实时生成内存缩略图，几百张 4K 原图同屏滚动丝般顺滑。
- 🖱️ **系统级整合**：附带自动化脚本，一键将“用图库查看”添加到 Windows 右键菜单，在任意文件夹右键即可呼出浏览器看图。
- 🎨 **专为创作者优化 (画师极度友好)**：
  - **沉浸式暗黑 Lightbox**：排除干扰，专注看图。
  - **无级缩放与自由拖拽**：鼠标滚轮缩放，左键按住拖拽找细节。
  - **快捷键支持**：左手放键盘（左右键切图，Esc退出），右手拿画笔，流畅工作。
  - **底部胶卷栏**：快速预览相册其他图片并随时跳转。
- 🧹 **一键隐私清理**：支持单独移除或一键清空所有已保存的文件夹记录（仅清空本程序记录，绝对不删源文件）。

## 🚀 快速开始

### 1. 环境准备
请确保你的电脑上已经安装了 [Node.js](https://nodejs.org/)。

### 2. 安装与运行
如果你是开发者：
```bash
# 1. 克隆本项目
git clone https://github.com/你的用户名/Local-RefImage-Viewer.git

# 2. 进入目录
cd Local-RefImage-Viewer

# 3. 安装依赖 (这一步很重要，用于安装 sharp 图像处理库)
npm install

# 4. 启动服务器
node server.js
```
启动后，浏览器会自动访问 `http://localhost:3000`。

**如果你是画师/普通用户（超简单）：**
1. 下载并解压本项目，确保你电脑安装了 Node.js。
2. 双击运行目录下的 **`一键启动(无黑框).vbs`**。
3. 它会在后台静默启动服务器，**没有任何黑框弹窗打扰你**，并自动为你打开浏览器！
4. 也就是点一下就能直接开始看图。想关闭时，双击 **`关闭服务器.bat`** 即可。

### 3. 如何集成到右键菜单？(Windows专属)
1. 在项目根目录下，双击运行自动生成的 `add_context_menu.reg` 文件。
2. 在安全警告中点击“是”。
3. 完成！现在你可以去电脑里找一个装满图片的文件夹，**右键点击空白处**，选择 **“用图库查看(&G)”** 即可体验！
*(如果你以后不想用了，双击运行 `remove_context_menu.reg` 即可干净卸载。)*

## 🛠️ 技术栈
- **后端**：Node.js + Express
- **前端**：HTML5 + Vanilla JS (原生 JS，无框架负担)
- **图像处理**：Sharp
- **系统交互**：PowerShell (呼出系统文件夹) + Registry (右键菜单)
